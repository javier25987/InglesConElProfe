# Clases de ingles con el profe

En esta pagina compilo todas las clases de ingles que he tenido con el profe y guardo todos los apuntes de estas.

> **NOTA:** en la barra lateral se despliega el menu para seleccionar la clase que se desea abrir

Todas estas clases estan en formato markdown y si se desea el codigo fuente de estos archivos se puede acceder mediante el link [GitHub](https://github.com/javier25987/InglesConElProfe) ahi tambien se encuentra el codigo python de la aplicacion usando el modulo streamlit, he qui un link a su documentacion [StreamlitDoc](https://docs.streamlit.io).
